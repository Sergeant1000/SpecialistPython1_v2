## "Кратные пяти из диапазона"

# Даны два целых числа.
# Вывести список всех чисел кратных пяти в диапазоне между заданными числами.

a = int(input("Первое число:"))     # Первое число
b = int(input("Второе число:"))    # Второе число

n = a

while n <= b:
    if n% 5 == 0:
        print(n)
    n = n+1
